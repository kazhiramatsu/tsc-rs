# tsc-rs

`tsc-rs` is a Rust port of the TypeScript compiler (tsc 6.0.3), focused on
reproducing `tsc` diagnostics byte-for-byte, verified by differential
testing against the real `tsc` oracle over the TypeScript conformance
corpus.

## Status

Milestones M1 (parser), M2 (binder), M3 (types & relations), M4
(checker skeleton + statement/expression/call extraction), M5 (flow
narrowing), and M6 (inference & overloads, incl. TS 5.5 body-inferred
type predicates and instantiation expressions) are complete; the

## Repository Layout

- `tsrs2/`: the codebase — a self-contained Cargo workspace with its own
  conformance corpus (`tsrs2/ts-tests/`) and pinned TypeScript oracle
  (`tsrs2/vendor/typescript-6.0.3/`).
- `docs/`: design documentation; `docs/design/greenfield/` is
  authoritative. Start at [docs/README.md](docs/README.md); setup and
  verification details are in [docs/setup.md](docs/setup.md).
- The paused v1 codebase is preserved at tag `v1-final`.

## Verification

All gates run from `tsrs2/` (toolchains are pinned:
`rust-toolchain.toml` at the repository root, `tsrs2/.node-version`
for the oracle):

```sh
cd tsrs2
cargo xtask ci            # full gate suite (must be green on main)
cargo xtask conformance   # conformance sweep (optionally --band 2xxx)
cargo xtask readme-status # regenerate the Status block above
```
